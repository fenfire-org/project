Johdatus xanalogiseen mediaan

Ted Nelssonin jo vuonna 1960 ideoima xanaloginen media saattaa tuoda tulevaisuudessa suuren murroksen tietokoneiluun. Mistä tekniikassa on oikeasti kysymys? Kuvitellaan jokaisen näppäiniskun tietokoneella olevan identifioitu, eli Pekan naputtelema A-kirjain eroaa merkittävästi Jukan näppäilemästä A-kirjaimesta. Tämä saattaa kuulostaa aluksi hyvinkin mahdottomalta, mutta ei sitä suinkaan ole - kunhan unohdetaan ainakin jossain mielessä ASCII merkistö aluksi.

Xanaloginen media voidaan toteuttaa seuraavasti. Talletetaan käyttäjän näppäilyn sijaan, esim. A-kirjain (0x41), sitä kuvaava identiteetti, joka viittaa ko. kirjaimeen. Ohjelmoijille sama voidaan ilmaista viitteellä, pointterilla, referenssillä tms. Tämän jälkeen talletetaan kirjain talteen kyseisen viitteen osoittamaan paikkaan.

Xanaloginen media, joka tässä esimerkkitapauksesa on tekstiä, antaa huomattavat edut verrattuna tavanomaiseen tekstiin. Kaikki lainaukset tekstistä sisältävät itsessään viitteen alkuperäiseen materiaaliin ja kirjoittajaan. Esimerkiksi lainattaessa tekstiä Keskisuomalaisesta vuodelta 1983 tammikuun toiselta sunnuntailta, voidaan käyttöliittymässä esittää viitattu artikkeli ja lehti.

Xanaloginen media ei pidä sisällään ainoastaan tekstiä, tai yksittäisiä näppäilyjä. Xanaloginen teksti on kuitenkin monesti ensimmäinen ajatus, joka xanalogisesta mediasta tulee mieleen. Esimerkiksi Fenfire käyttää (tieteellisten) dokumenttien tarkasteluun kuvia kustakin sivusta. Muita medioita voi olla esim. ääni ja video. On huomattu, että vaikka puhutaan useista medioista, niitä oikeasti on huomattavan vähän kongretisoitavissa.

Seuraavassa on esimerkki xanalogisesta tekstistä Fenfire ympäristössä. Käytännössä xanaloginen media esitetään ortogonaalisesti RDF graafissa laittamalla XML tieto tyypitetyn literaalin sisälle. 


# An example of use of xanalogical text with TIML implementation.


import org.nongnu.alph as alph

textMerkki = alph.TString.newRICC("merkki")
textEsi = alph.TString.newRICC("Esi")


text = textEsi.plus(textMerkki)
print text
print text.toXML()

text = textMerkki.plus(textEsi)
print text
print text.toXML()
